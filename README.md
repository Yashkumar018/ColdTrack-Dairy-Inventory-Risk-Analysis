# ColdTrack-Dairy-Inventory-Risk-Analysis
ColdTrack: Dairy Inventory Risk Analysis  Power BI dashboard to analyze and reduce inventory risks in the dairy cold chain. Identifies delivery delays, expiry patterns, overstocking &amp; stockouts across major brands. Delivers root cause insights, reorder recommendations, and estimated savings of ₹1.41 Lakh.  Built with Power BI, SQL &amp; Excel.
